# MeNabung UI

AI DeFi advisor that helps you grow IDRX savings on Base.

## Quick Start

```bash
npm install
cp .env.example .env.local  # Add your API keys
npm run dev
```

## Environment

```env
OPENAI_API_KEY=
NEXT_PUBLIC_ONCHAINKIT_API_KEY=
```

## Stack

Next.js 16 · OnchainKit · Tailwind CSS · motion/react
